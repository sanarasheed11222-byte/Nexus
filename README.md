# Nexus - Investor & Entrepreneur Platform

Live Demo: https://nexus-jrwz.vercel.app
Backend API: https://nexus-cyan-delta.vercel.app

## Tech Stack
Frontend: React, TypeScript, Tailwind CSS, Vite
Backend: Node.js, Express, MongoDB, Socket.IO, JWT

## Features
- Authentication & role-based profiles
- Meeting scheduling with conflict detection
- Video calling (WebRTC + Socket.IO)
- Document upload with e-signature
- Real-time messaging
- Payment simulation (deposit/withdraw/transfer)
- Notifications system
